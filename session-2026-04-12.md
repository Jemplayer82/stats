## Project state as of this session

**Usage Meters** is a Flask dashboard (`app.py`) that shows live usage for:
- Claude.ai (session cookie → `/api/organizations/{org}/usage`)
- Ollama.com (BeautifulSoup scrape of `ollama.com/settings`)
- Gemini (Google Cloud Monitoring API, service account JSON)
- Proxmox cluster (nodes + VMs/containers via PVE API token)

Config stored in SQLite via `AppConfig` key-value model. Runs as a systemd service (`usage-dashboard.service`).
